Sistema de Automação

Realizar controle de sistemas prediais não é necessariamente uma novidade, porém, o que encontramos no mercado, são sistemas pouco eficientes, que funcionam como um liga/desliga remoto, e sem trazer informações para o cliente, que muitas vezes instalam sistemas paralelos para ter as informações que precisam, como medição de energia.
Defendemos a aplicação da boa engenharia para desenvolvimento de um projeto integrado, tornando o sistema de BMS, um sistema de gestão predial, onde além do funcionamento regular dos sistemas, como controle de ar condicionado, iluminação, energia e utilidades, o software permite a extração de dados para serem utilizados no aperfeiçoamento da operação, com indicadores que auxiliam a gestão, na busca de melhores resultados.
